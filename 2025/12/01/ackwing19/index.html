今天的模拟赛一共有 n 名学生参加，包含 m 道题。学生的编号范围是 1∼n，学生 i 在题目 j 的得分为 ai,j。
小黄是其中的学生 x。当某位学生在某道题目获得了该题的最高分或最低分（包括并列最高分或并列最低分）时就有可能被选中上台讲题。
小黄准备篡改自己不超过 c 道题目的得分。请你告诉她，她是否可以通过这样的方式，使自己完全不可能被选中上台讲题呢？

输入格式:
本题包含多组测试。
第一行一个整数 T，表示测试点中的测试数据组数。
对于每组测试数据：
第一行三个整数 n,m,x，表示学生数、题数和小黄的编号。
接下来 n 行，每行 m 个整数，第 i 行的第 j 个整数表示学生 i 在题目 j 的得分 ai,j​。
最后一行一个整数 c，表示小黄最多可以篡改的题目得分数量。

输出格式:
一行一个字符串，up 或 down。
up 表示小黄仍然有可能被选中上台讲题，
down 表示小黄不可能被选中上台讲题。

输入样例1:
2
3 3 1
40 50 50
60 40 50
70 45 55
3
3 3 1
50 50 50
60 40 55
70 45 50
2
输出样例1:
down
up
#include
#include
#include
#include
using namespace std;
void dunc() {
int n, m, x;
cin >> n >> m >> x;
x–;
vector<vector> scores(n, vector(m));
for (int i = 0; i < n; i++) {
for (int j = 0; j < m; j++) {
cin >> scores[i][j];
}
}

int need_change = 0;

for (int j = 0; j < m; j++) {
    int x_score = scores[x][j];
    bool is_min = true, is_max = true;

    for (int i = 0; i < n; i++) {
        if (scores[i][j] < x_score) {
            is_min = false;
            break;
        }
    }

    for (int i = 0; i < n; i++) {
        if (scores[i][j] > x_score) {
            is_max = false;
            break;
        }
    }

    if (is_min || is_max) {
        need_change++;
    }
}

int c;
cin >> c;

if (need_change <= c) {
    cout << "down";
}
else {
    cout << "up";
}

}
int main() {
int T;
cin >> T;
while (T–) {
dunc();
if (T != 0) {
cout << ‘\n’;
}
}
return 0;
}
